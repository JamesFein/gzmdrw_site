智者当借力而行：浅析六大 Web3 任务平台

2025年02月10日 • web3.0 •

随着 Web3 行业的发展，用户和项目方数量都在不断快速增长，竞争也越来越激烈。如何在竞争中胜出，选对工具是关键，正所谓工欲善其事必先利其器，智者当借力而行。无论用户和项目方学会利用好 Web3 任务型平台工具，都将事半功倍。

一、web3 任务平台简介

web3 任务类平台主要业务是提供关注项目方社交账号、答题、链上操作以及任务悬赏等任务，用户完成后可获得积分、NFT 以及 token 奖励。Web3 任务平台起到了 Web3 入门教育，以及链接用户与项目方的作用。Web3 任务类平台起始于 2020 年 Web 2 的社交媒体营销服务商 Gleam.io 开始为 web3 行业提供服务，其后 web3 任务型平台也大量涌现，Web3 任务型工具经过两三年的发展，已经出现了上百个项目，并且正在成为加密世界的新流量中心。

二、用户如何利用 web3 任务平台

  • web3 任务平台一般用户体验做的相对较好，有的平台做的还比较有趣，操作也相对更简单容易，所以对于 web3 新人可以做为链上操作入门，学习 web3 行业基础知识和了解学习项目或社区。

  • 对于资深 web3 从业者，通过 Web3 任务管理平台，可以在同一平台上接触并了解到多类项目和社区，有些平台还有项目或社区的分类和筛选功能，用户可以省去很多时间和精力，寻找到自己感兴趣的优质项目。

  • 对于撸空投挣收益的用户，通过做 web3 任务平台上的任务可以获得相应奖励，如 NFT、POAP、空投资格、代币奖励等，投入的成本以及风险相对较低，是一个功能高级的空投聚合类网站。

  • 此外用户通过在 web3 任务平台上做任务可以积累自身的链上和链下的活动证明，有利于建设个人的 Web3 身份，打造个人 web3 声誉。

三、项目方如何利用 web3 任务平台

  • 项目方可以通过发布任务并给予奖励的形式，吸引平台上的用户参与以达到宣传目的。对于发展初期且粉丝量不高的新项目而言，借助于 Web3 任务管理平台的流量效应，可以实现相对更好的营销效果。对于成熟项目方可以多做激活和留存类活动，也可以做拉新活动。

  • 项目在 web3 平台上发布任务相对更简单及时有效，项目方可以快速完成一个效果显著的 Web3 运营活动，为自己的项目和产品增加用户量与使用量。

  • 很多 Web3 任务平台有用户的链上链下数据分析功能，项目方可根据任务平台的数据筛选出真实用户并发放奖励，做到防止空投给无效用户和以及预防女巫攻击，做到精准投放精准营销。

  • 利用 Web3 任务平台项目方可以与其平台上他项目或社区发起多项目的联合市场活动。

  • Web3 任务平台都漂亮的、多维度、精细化的数据仪表盘展示面板，项目方可以了解自己本次活动的效果好坏（转化率）、触达的用户画像是否精准（活跃度）、活动任务设置是否合理（每个任务的完成率）等等。可以动态调整运营策略。

四、六大 web3 任务平台

当前 Web3 任务平台工具可谓是百花齐放。从共同点来说，它们都是任务型营销工具，都是 web3 用户增长工具。但各平台的切入点和侧重点还是有一定的差别的，按照平台的切入点可分为流量类、教育类以及细分垂直类三类。根据业务模式侧重点的不同，平台可以分为偏向 to C、to B、to C+to B 三种。VeDAO 从赛道中选取了 6 个代表项目给大家做个简单介绍。

4.1、Galxe



Galxe，原名 Project Galaxy，是目前最大的 Web3 任务平台。截至 2023 年 6 月 24 日，已与超过 2700 个项目方 / 社区合作，推出超过 10000 个任务，拥有超过 1100 万 Galxe ID 的用户。Galxe 允许项目方在平台上发行 NFT，满足条件的用户可获得 Mint 资格，项目方还可以通过 平台的数据凭证提供定制化服务。用户可以在平台上看到各个项目发布的任务，并能够从中寻找潜在空投、参与社区活动、发现新项目。

Galxe 搭建了一套自己的数字身份凭证 Galxe ID，以及从链上标记 OAT。其通过对链上数据的分析，给不同用户打上了各种标签，对于每个 Quest，也将产生一个 Credential 集合，作为白名单导出到第三方进行应用（如 Token 派发）。

Galxe 推出的任务多为特定项目任务，主要为关注项目方推特，转推，加入 Discord，链上交互等。用户完成任务获得的奖励主要为 OAT（链上成就 NFT）和其他 NFT，其中部分 NFT 可以售卖。任务多且较为繁琐，不适合 Web3 入门教育，但适合空投猎人和进阶选手积累链上交互记录。平台的商业模式偏向 to B，属于流量类平台。



Galxe 也有实人认证模块：Galxe Passport，用户提供 KYC 信息后可铸造一个 SBT，这也是 Galxe 在防女巫方面的措施。Galxe Credentials 是用户交互凭证，通过读取用户链上链下交互记录，自动授予给用户。

4.2、Zealy（原 Crew3）



Zealy 是一个侧重于对社区进行宣传的平台，主要目标是帮助 web3 社区不断成长。Zealy 以 Web3 Discord 作为其发展定位，并具备较高的社交属性。社区内活动模板较多，包括日常打卡、内容创作、邀请任务等运营任务。相对于其他平台，Zealy 更聚焦且分类十分清晰，该平台还具备积分排名系统。在 Zealy 中，部分项目任务需要写文、截图证明、邀请等方式才能完成，以过滤很多自动化羊毛党，保证用户真实性。

平台提供的任务基本都是特定项目任务，用户通过关注项目方推特，转推，加入 Discord，注册项目账号，项目知识答题等任务。任务多为项目方营销获客设计，因此项目的商业模式也是偏 to B 的, 也属于流量类平台。

4.3、QuestN



QuestN( 原 Quest3) 是一个由 Hogwarts Labs 孵化的 Web3 任务活动平台，是 GRT14 捐赠项目之一。平台是一个以服务长尾 B 端用户为特色的 Web3 任务凭证平台。在大部分主流任务凭证平台，如 RabbitHole，Galxe，只有具有一定规模的 B 端客户才能通过平台的筛选发布任务，而 QuestN 不设置门槛，小 B 端客户如 KOL，社区，DAO 等都可以发布任务，但平台为了保证 B 端客户的质量，也要求绑定官网，社交账号等，对于提供 token 奖励的任务，还要求 B 端客户提前将奖励 token 存入金库，起到验资的作用。

QuestN 上提供的任务基本都是特定项目任务，用户通过关注项目方推特，转推，加入 Discord，注册项目账号等任务，提供的奖励有 token，NFT，白名单等。任务多为项目方营销获客设计商业模式也是偏 to B 模式的， 也属于流量类平台。

4.4、Layer3



Layer3 设计了一套“New to Web3”系列任务，由浅入深地引导用户完成 Layer2，DAO，DeFi，NFT 等领域的链上交互，完成后可获得 NFT 和经验值奖励。完成任务获得的 NFT 也可出售，但普遍价格不高。同时，也提供了特定项目任务，主要操作为链上交易，质押，添加流动性等，但不同于 Galxe，这些任务难度较低一些，且大多没有最低交易额要求，进行价值 1U 的交互也可完成任务，并且也提供了任务指导，因此，对不想花费太多资金的 Web3 初学者比较友好。项目方主要是为了获得客户，也会与各类交互平台合作。项目方需要向平台方支付一定费用，并向用户提供一定激励，可能是空投也可能是其他激励，所以平台的商业模式是 to B+to C 结合模式。可归于教育类项目。

4.5、RabbitHole



RabbitHole 是一家老牌的去中心化的链上激励交互平台。当前产品分为 web3 新用户教育平台 Skill 和广告投放 Quest 两类。一类是 Skills，目前有四种 Skills，分别是 Layer2，DAO，DeFi，NFT。需要完成的任务多为基本的 Web3 链上操作。其中每个任务的 Details 中都提供了详细的教程。完成这些任务后，用户将获得一个不可转让的“Intro to L2” NFT。

另一类任务是 Quests，这些多为单个项目方发布的交互任务，除了奖励 NFT，有的 Quest 还会奖励项目代币，但奖励项目代币的任务通常需要 BrightID 实人认证防女巫（女巫攻击即为一个用户伪装成多个身份进行操作）。

总体而言，RabbitHole 提供了较为详细的 Web3 链上操作入门教育，适合小白用户使用，合作项目方较弱。RabbitHole 曾在 2022 年 2 月份宣布完成 1800 万美元 A 轮融资，由 Greylock 和 TCG Crypto 领投。商业模型偏 to C, 可归为教育类项目。

4.6、TaskOn



TaskOn 是一个 Web3 任务协作平台，致力于 Web3 项目的营销和运营增长。TaskOn 是 Moonbeam 系列应用的一部分。平台能够提供一些自动化功能，减少操作流程。例如，API-Verified 可以帮助自动验证参与者是否进行产品特定操作。该平台还具备自动发奖功能，可以在活动结束后代替活动方立刻发奖，而活动方所有的操作只有创建活动，将足够的奖励（Token 或 NFT）充进自己的 TaskOn 账户；TaskOn 还可以替活动主办方创建活动，进一步减少主办方的操作。在当前的 Web3 任务平台中，TaskOn 是功能精简易用，发展速度较快的平台之一。该平台因具备一系列防机器人措施，是目前真实用户率最高的平台之一（高达 92.9%）。平台商业模式偏 to B, 属于流量类平台。
