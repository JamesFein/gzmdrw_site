BRC20 真的懂 Web3？探寻宗教式社区建设，Yuga Labs「猿」宇宙的案例解析

2025年02月05日 • NFT •

撰文：彭紫盈，欧科云链

早前，NFT 领域最强 IP 缔造者 Yuga Labs 官宣：将基于 Ordinal 协议在比特币区块链上推出 NFT 系列「TwelveFold 」，为比特币 NFT 生态添加了催化剂。所以以太坊失宠了吗？事实上，据欧科云链 OKLink 多链浏览器数据显示，继 5 月 8 日 BTC 挖矿手续费收益创近 5 年来新高，达到 683.36BTC（约 1908 万美元），显然，BRC20 高的不仅是热度，链上的拥挤度也不容小觑。那么一个强大的社区生态该如何打造？除技术外，本文将通过在美国 Consensus 共识大会对 Yuga Labs 现场的观察与交流，梳理和分析 Yuga Labs 是如何一步步在火热的争斗中成为社区霸主。

不可否认，传统的科技巨头在「下一代互联网」的豪赌中输得满盘皆落索，而加密货币生态圈的崛起则引领着这一时代的变革。比起相对稚嫩的 BRC20，以太坊拥有着无可匹敌的优势，而 Yuga Labs 则是以太坊生态系统中最杰出的代表之一。正如其新任 CEO Daniel Alegre 说在共识大会 Consensus2023 中说的：「我们不是一家游戏公司，而是一家内容（content）与社区（community）公司」。其前瞻性的宗教式思维在游戏、NFT 和元宇宙领域迅速崛起，成为无冕之王。

数据不说谎。先来看看 Yuga Labs 的近期收获

Yuga Labs 16 个 NFT 收藏品占整个 NFT 行业的 34.6%，过去 2 个季度的总交易量为 20 亿美元。



Yuga Labs 于 2021 年首个推出的由 1 万只无聊猿组成的 NFT 系列 —— Bored Ape Yacht Club 在 NFT 行业中拥有最高的地板价（NFT 系列中市场定价最低的那个价格），价格高达为 9.6 万美元，紧随其后的是 CryptoPunks，价格为 9.2 万美元。



Otherdeed for Otherside（Otherside 是汇集了各种 NFT 项目的元宇宙游戏世界；Otherdeed 是为该游戏设计的全新 NFT，是某种虚拟土地的凭证）是表现最好的元宇宙 NFT 集合，在过去两个季度的交易量为 3.22 亿美元，平均销售价格为 3112 美元。



$APE（ApeCoin，又叫猿币，服务于无聊猿社区及相关社区的治理和实用代币）交易总额已突破 100 万 ETH，达 16.8 亿美元市值，使其成为领先的元宇宙游戏代币。

然而这些耀眼的数据背后，始于两个并不起眼的「小」人物。

两个「无聊」的人创造的无聊猿

Yuga Labs 的最初创始人，一个是长期病患 Wylie Aronow，另一个是无名作家 Greg Solano，当时两人空有一腔文学抱负，却无处施展。他们「猿」分的开始，起源于一条短信：「嘿，我们一起做一个 NFT 吧」。而无聊猿的创意来源，是一个关于财富自由的想象。两人构思出一个俱乐部，决定以币圈一群独特的人群作为 NFT 的主角，因为无脑投资而发达（to Ape in），一夜之间成了既有金钱也有时间的人。他们大多面无表情，戴着各式的帽子和墨镜，穿着花里胡哨的外套，有的叼着雪茄，有的咬着披萨。他们的娱乐活动就是在俱乐部的浴室墙壁上任意涂鸦。对应着，这些 NFT 在现实世界的持有者可以进入虚拟创作空间 BAYC bathroom，在那里进行数字创作，在墙壁上写写画画。在这些在思想和概念的碰撞后，无聊猿诞生了。之后两位创始人聘请了五位艺术家来设计猿类特征，并加入算法，生成一万个 NFT。



直到今天，Yuga Labs 仍在一步步扩大其帝国版图。正如他们的 Pitch Deck（融资简报）中写的：「BAYC 只是个开始，我们正在建设下一个边界」

无聊猿的爆火并非侥幸。正如 CryptoPunk 创始人在被收购之后对媒体所说：「我们的专长是在技术早期做创造，不擅长社群管理和维护公共关系。Yuga Labs 是头像类 NFT 的创新者，他们更擅于项目运营。」以下将从 Yuga Labs 独特的商业手段、强大的社区营销及理念共识建设三方面分析，他是如何一步步从 Web3，走向 Web2，最后到传统世界的。

「让一部分人先富起来」的商业手段

除了 Yuga Labs 本身赚的盆满钵满，对于持有无聊猿的人，他们更是大手笔让利，把空投计划贯彻始终，知行合一，与社区共赢也是其核心团队在共识大会无论哪个会场上反复强调的思想。根据 OpenSea 的最高单价记录，无聊猿 NFT 持有者最高获得价值 280000 美金的空投奖励。他们首先推出了 1 万个无聊猿的「宠物同伴」NFT——无聊猿犬舍俱乐部（Bored Ape Kennel Club，BAKC)，免费发放给 BAYC 的持有者。随着热度不断攀升，这只狗的售价一度超过猴子，目前地板价维持在 52ETH。

其后继续推出衍生品——变异猿游艇俱乐部（Mutant Ape Yacht Club，MAYC）这批 NFT 是第一版无聊猿的「基因突变版」。团队向 BAYC 持有者空投带有突变基因的「血清」，具体分为 M1、M2、M3 从低到高的三个等级。不同等级的血清决定着原版无聊猿基因突变的特征和程度，如果拿到 M3，意味着可以获得非常稀有和独特的变异猿猴，市场价也更高，其中 Mutant Serum 变异血清 NFT 就曾以 888ETH 的天价售出。

随后相继面世的代币 $APE 和汇集了各种 NFT 项目的元宇宙游戏世界 Otherside，在公布时也给 BAYC/MAYC NFT 的持有者预先留了一席之地。近期，在他们最新推出的游戏中，Otherdeed 土地持有人都可以认领到一艘船只——Vessel，预定申领时间在 4 月初，每个 Vessel 都是一个单独的 ERC-721 NFT，会在申领过后空投到 Otherdeed 持有者的钱包中。



相比起一众 NFT 项目方单纯的销售模式，Yuga Labs 追求的更是一种双赢局面，他把 NFT 当成一种赋能运动，让用户源源不断地享受红利，形成了另类的财富效应。就像他们新任 CEO Daniel Alegre 说在共识大会 Consensus2023 中说的：「我们要做的不是赚用户的钱，而是要一起赚钱」。

我的是你的，你的还是你的

如果说「赋能」是 NFT 价值的关键，而「分享」则是 Yuga Labs 的基因。

Yuga Labs 打造了自己的商业变现模式，这也是 BAYC 不同于其他 NFT 的独特之处，一个在目前被印证可行的财富密码。通过将自己的知识产权进行切割，对知识产权 IP 进行分散的另类合作模式，而这种 De-Collab（去中心化协作模式）也在 2023 共识大会上被核心团队多次强调。BAYC 授予了其 NFT 购买者一系列广泛的权利，他们可以再不必顾及维权的情况下对 BAYC 进行任意商业使用。而这种许可证授予是无界限的，用户可以复制以创造基于 BAYC 艺术品的衍生作品。之后在买下 Larva Labs 手上的 CryptoPunks 和 Meebits 之后，Yuga Labs 也立马将商业权限交还给持有者。这对于传统的授权来说是史无前例的。

从此消费者不仅仅只是消费者，更是参与者、拥有者和创业者。

于是大量延伸的作品很快诞生了。穿的、玩的、看的、甚至是还有吃的，涵盖娱乐、音乐、时尚、游戏和虚拟世界在内的多个领域。团队还资助其中使用猿猴 IP 的创业者，鼓励他们发挥创造力，扩大品牌影响。而这种自下而上的方法，让每个使用猿猴形象做自己商业产品的人同时参与 Yuga 的品牌建设。一众社区成员都在通过各种方式给这个社区和品牌赋能，就像一个社区版的 Supreme，创造着自己的文化。



根据 Bitpush News 的资料显示，Yuga Labs 的 IP 生态系统快速增长：数量从 22 年 8 月份的 80 增长到现在的 200+。



图片来源：Bitpush News

正如创始人 Wylie Aronow (Gordon Goner）所说：「人们用猿创造的任何东西只会让品牌成长」。确实，无聊猿无疑是目前唯一把这套独有的营销模式玩通，还玩得很大的赢家。而在他成功崛起的背后，植根于人们对其品牌文化的喜爱和社区认可，这种高度的共识和价值观认同才是关键。

持有无聊猿，你就是大佬

就像劳力士、就像法拉利，BAYC 在 Web3 界已经发展成为一种地位的象征。是否拥有或是否有能力拥有 BAYC 似乎已经成了行业内鉴定财力的一种标准。随着无聊猿二级市场价格的水涨船高，现在如果你的 Twitter 头像是一张无聊猿的 NFT，那么你的自豪感不亚于开着迈巴赫在街上了。

BAYC 的高速发展离不开社区的作用，自发行以来，在短时间内迅速成长成了一个庞大的社区。对于持有 BAYC 的玩家来说，这不仅是收藏品，更是一个俱乐部。BAYC 成了加密爱好者的潮流代表，他们相信这一领域，并看到该技术在艺术、音乐、游戏和娱乐等行业的许多应用。这种信仰使得人们愿意加入一个拥有相同价值观的社区。而同时又满足了用户心中的归属感。BAYC 的故事背景源于许多 NFT 收藏家的心声：在加密市场中投资者在 NFT 而实现了财富自由的梦。

同时，他们也把 BAYC 的理念共识从线上拉到线下，从 Web3 推向传统。Yuga Labs 举办了多场线下活动，只有无聊猿的持有者才能够参加，这无疑是一场彰显身份地位的派对，也是一个加密精英圈子，众多大佬都是这个俱乐部里的成员，让无聊猿的价值得到了极大的提升。Yuga Labs 相继举办了互动寻宝活动、线下 Ape FEST、BAYC x MAYC 手游大赛等等。



Ape FEST 、无聊猿派对，图片来源：NFT Evening

当然，支撑起 BAYC 关注度持续不减的关键在于明星的加盟，如歌手贾斯汀比伯、网球名将塞雷娜·威廉姆斯、前 NBA 球员奥尼尔等，有了一众名人加持，这让 BAYC 这名字从未淡出过人们视野中的同时，更进一步地突破了用户圈层。

写在最后

总的来说，不论是共识大会上还是笔者本次的案例分析，Yuga Labs 的核心是给人一个「共富」的愿景并且不断用「赋能」与「分享」来实现，从他们的历史故事到商业模式再到共识建立，从 BAYC、 BAKC 和 MAYC 的 NF，到代币 $APE，再到元宇宙 Otherside，都是再不停增强其 IP 能量。核心是给用户一个足够吸引人的故事，让用户用自己的脑袋思考这个故事该走向何方，在途中又该为其作出些什么贡献。

他们创造的这种新玩法，不仅可以成为加密界的引路人，更可为传统商业提供启示，比起通常依赖于产品或服务的销售， Yuga Labs 的商模式则更注重创造和维护 IP，以及与社区的互动和合作。这种基于社区的协作和共创，是传统商业能够借鉴的，并引发他们思考如何在知识产权分散、共创生态的背景下，发掘新的商业价值。

而 Yuga Labs 的成功，离不开以太坊这块强大的基石。作为一个先进的生态系统，以太坊 NFT 社区在铸造、交易、存储和展示 NFT 方面拥有更成熟和高效的工具和解决方案，为 Yuga Labs 的每一步提供更多的灵感。同时，以太坊生态中有众多活跃的开发者社区，为整个生态系统的发展作出了重要贡献。相比之下，BRC20 还处于起步阶段，但站在巨人的肩膀上才能看得更远，目前正是循着以太坊的脚步在其生态中发现更多的机会和发展空间的大好机会。

在万马齐行的赛道上，Yuga Labs 能否独领风骚，成为杀出重围的黑马？而 BRC20 又能否奋起直追，在这场角逐中站稳脚跟？让我们且行且期待。
